# 🌐 StreamBit Web UI - دليل الاستخدام

## واجهة ويب جميلة لمعالجة الصور!

---

## 🚀 كيفية التشغيل

### الخطوة 1: افتح PowerShell
```powershell
cd C:\MAMP\htdocs\flowxtra\streambit
```

### الخطوة 2: أضف Cargo للـ PATH
```powershell
$env:Path += ";$env:USERPROFILE\.cargo\bin"
```

### الخطوة 3: شغل الخادم
```powershell
cargo run --package streambit-web-ui
```

**أو للأداء الأسرع:**
```powershell
cargo run --release --package streambit-web-ui
```

### الخطوة 4: افتح المتصفح
انتقل إلى: **http://localhost:8080**

## 📊 فهم النتائج

### عدد الصور
عدد الصور التي تمت معالجتها بنجاح

### الوقت (ميلي ثانية)
الوقت الكلي المستغرق للمعالجة
- أقل = أفضل
- Debug mode: ~30-50ms لـ 3 صور
- Release mode: ~5-10ms لـ 3 صور

### صورة/ثانية (Throughput)
عدد الصور المعالجة في الثانية
- Debug mode: ~80-100 صورة/ثانية
- Release mode: ~400-800 صورة/ثانية

### Shape
شكل الـ Tensor الناتج:
- `[3, 224, 224]` = 3 قنوات (RGB) × 224 عرض × 224 ارتفاع

### مثال 3: مقارنة الأداء
1. جرب في Debug mode

## 🔧 حل المشاكل

### ❌ "error: no matching package named `actix-web`"
**الحل:**
```powershell
cargo update
cargo run --package streambit-web-ui
```

### ❌ "Address already in use"
**الحل:**
المنفذ 8080 مستخدم. أوقف البرنامج الآخر أو غير المنفذ في `main.rs`

### ❌ الصفحة لا تفتح
**الحل:**
- تأكد أن الخادم يعمل (شوف رسالة "Server running")
- جرب http://127.0.0.1:8080 بدلاً من localhost
- تأكد من Firewall لا يمنع الاتصال

### ❌ "Failed to process images"
**الحل:**
- تأكد أن الصور بصيغة صحيحة (JPG, PNG, WebP)
- تأكد أن الصور ليست تالفة
- جرب مع صور أصغر حجماً

## 🎨 تخصيص الواجهة

### تغيير الألوان
عدل ملف `streambit-web-ui/static/index.html`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
غير `#667eea` و `#764ba2` للألوان المفضلة

### تغيير حجم الصور المعالجة
عدل ملف `streambit-web-ui/src/main.rs`:
```rust
match processor.load_batch(temp_files.clone(), Some((224, 224)), None) {
```
غير `(224, 224)` للحجم المطلوب

### إضافة أوضاع Resize
عدل `ResizeMode::Bilinear` إلى:
- `ResizeMode::Nearest` - الأسرع
- `ResizeMode::Bicubic` - جودة أفضل
- `ResizeMode::Lanczos3` - أفضل جودة

## 📱 استخدام من أجهزة أخرى

### الوصول من جهاز آخر في نفس الشبكة:

1. اعرف IP الخاص بك:
```powershell
ipconfig
```
ابحث عن IPv4 Address (مثلاً: 192.168.1.100)

2. غير في `main.rs`:
```rust
.bind(("0.0.0.0", 8080))?  // بدلاً من 127.0.0.1
```

3. افتح من الجهاز الآخر:
```
http://192.168.1.100:8080
```

## 🚀 نصائح للأداء الأفضل

1. **استخدم Release mode دائماً للإنتاج:**
   ```powershell
   cargo run --release --package streambit-web-ui
   ```

2. **عالج دفعات كبيرة:**
   - 10-50 صورة أفضل من صورة واحدة
   - المعالجة المتوازية تستفيد من الدفعات الكبيرة

3. **راقب استخدام CPU:**
   - يجب أن يكون 100% على كل الأنوية
   - إذا كان أقل، زد عدد الصور

4. **استخدم صور بنفس الحجم:**
   - أسرع في المعالجة
   - أقل استهلاك للذاكرة
